# Feature Inventory

This list reflects the actual system state and separates current features from
planned expansions.

## Current Features
- **Google Places lead search** with candidate generation.
- **Lead enrichment pipeline** with ethical web crawling.
- **Business intelligence extraction** with evidence and confidence.
- **AI readiness scoring** with multi-pillar breakdown and recommendations.
- **Policy-gated outreach** (DNC, frequency, time, channel rules).
- **Vapi call execution** for voice outreach.
- **n8n follow-up execution** for email or workflow automation.
- **Dashboard UI** for lead intelligence visibility and workflow oversight.
- **Multi-agent squad model** with isolated workflows per squad.
- **Audit trail** for decisions, policies, and memory updates.
- **SSRF-safe web fetching** and data sanitization for logs.

## Squad Management (Current)
- **Create and configure squads** for distinct market segments.
- **Assign agents per squad** with scoped roles and channels.
- **Squad-scoped policies** and workflow governance.
- **Isolated memory and audit trails** per squad.

## Engineering Highlights (Current)
- **Deterministic orchestration:** all actions flow through a single control plane.
- **Schema-constrained intelligence:** structured outputs enable safe automation.
- **Policy-first autonomy:** every outreach action is gated with rationale.
- **Replayable workflows:** audit logs allow reconstruction of decision paths.

## Planned Features
- **Autonomous outreach sequencing** with human escalation.
- **Appointment booking** and calendar synchronization.
- **Client onboarding** (intake, policy configuration, document workflows).
- **Multi-channel outreach** (SMS, LinkedIn, social channels).
- **Finance and operations** (invoicing, payment follow-ups, reporting).
- **Outcome learning** (message optimization, conversion analysis).
- **Multi-tenant deployment** and role-based access controls.
- **Agent marketplace** for custom agent strategies.
